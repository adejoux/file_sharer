This little application is used to share files.

Install
-------

1) clone project :

    # git clone https://github.com/adejoux/file_sharer.git

2) install gems
    # cd file_sharer
    # bundle install

3) Init sqlite database

    # rake db:setup

4) start rails

    $ rails s
